Technology
Built on a modern,
scalable stack
Enterprise-grade infrastructure powering real-time price data at scale.
Frontend
React Native + Next.js
Cross-platform iOS & Android app from one codebase. Next.js for fast server-rendered web.
Backend
Node.js + FastAPI
RESTful APIs on Node.js. Python FastAPI for ML-based price anomaly detection and deal scoring.
Database
PostgreSQL + Redis
Postgres for product & user data. Redis for sub-100ms search caching. InfluxDB for price time series.
Data
Rainforest + SerpAPI
Retail price feeds from 500+ sources. Google Shopping data. Custom scrapers for remaining sources.
Infrastructure
AWS + Cloudflare
ECS containers, RDS, ElastiCache, Lambda for alert jobs. Cloudflare CDN globally.
Payments
Stripe
Subscription billing, free trial management, and affiliate revenue tracking built in.
Notifications
Firebase + SendGrid
Push notifications via Firebase Cloud Messaging. Email alerts via SendGrid with personalization.
Maps & Tax
Google Maps + TaxJar
Store locations, directions, and hours via Google Places API. Accurate sales tax by ZIP via TaxJar.
